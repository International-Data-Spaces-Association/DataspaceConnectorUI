{{title}} The dashboard shows statistical data of the current connector. Overview of all currently offered IDS resources.

The table can be filtered and searched. New IDS resources can be added by clicking the "ADD RESOURCE"-button.

Existing IDS resources can be changed by clicking the pencil button or deleted by clicking the trash button. The information needed to add a new IDS resource is separated into three types:

Meta data: General information about the resource data.

Policy: Information about how often or how long the resource data can be used.

Representation: How is the data accessed?

Brokers: To which brokers should the resource be registered? Editing an IDS resource is separated into three types:
